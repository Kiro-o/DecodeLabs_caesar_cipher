# Caesar Cipher Tool

For Project 2, I created a Python tool that encrypts and decrypts messages using the classic Caesar Cipher method. 

## How it works:
- It takes a text and a shift value (between 1 and 25).
- It shifts the letters mathematically using ASCII values and modulo 26.
- It keeps uppercase and lowercase letters exactly as they are.
- It doesn't touch spaces or punctuation marks, so the message structure stays safe.

## How to run it:
Run `python caesar_cipher.py`, type your message, enter a shift number, and it will show you the encrypted text and then decrypt it back to prove it works.
